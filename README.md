# Programming Level II

This repository contains the source code for the Programming Level II course. The course covers Java basics to Data Structures and Algorithms (DSA) with Java.

## Contents

1. Java Basics
2. Object-Oriented Programming in Java
3. Data Structures in Java
4. Algorithms in Java
5. Advanced Java Concepts

## Project Structure

The project is structured as follows:

- `src/main/java/org/example`: This directory contains the main Java source files.
- `pom.xml`: This is the Maven project file that includes project dependencies.

## Setup

To set up the project, follow these steps:

1. Clone the repository: `git clone https://github.com/mehedithedev/Programming-Level-II.git`
2. Navigate to the project directory: `cd ProgrammingLevelII`
3. Build the project: `mvn clean install`

## Running the Application

To run the application, use the following command:

```bash
mvn exec:java -Dexec.mainClass="org.example.Main"
